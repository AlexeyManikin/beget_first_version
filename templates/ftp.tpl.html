Управление FTP аккаунтами
     На вашем тарифном плане доступно {FTPALL} дополнительных FTP аккаунтов.

    Дополнительные FTP аккаунты используются для разграничения доступа к различным разделам сайта или различым сайтам. Созданные аккаунты будут иметь доступ только указанным папкам и всем вложением в нее. При удалении FTP аккаунта папка на которую он ссылается не удаляется.

    Создание FTP аккаунта

Для создания FTP аккаунта Вам необходимо указать Имя пользователя, которое формируется в виде {LOGIN}_(логин). Логин должен состоять только из букв латинского алфавита, цифр и заков подчеркивания. В качестве директории пользователя укажите любую директорию относительно корня аккаунта. Если директория не существует - она будет создана.

Имя пользователя
Путь к дериктории
Пароль аккаунта
